🚢 La Herradura SPA - Portal de Envios Chile-Cuba 🇨🇱 🇨🇺
Este es el portal oficial de La Herradura SPA, una plataforma diseñada para informar y asistir a los clientes en sus procesos de envio de carga desde Chile hacia Cuba. La aplicacion centraliza informacion critica, tarifas y herramientas de rastreo en un solo lugar para mejorar la transparencia y comunicacion con el cliente final. El proyecto esta construido sobre Next.js y optimizado para una carga ultrarrapida.

🚀 Caracteristicas Principales
📦 Rastreo Multi-Sitio: Integracion directa con los principales portales de logistica como Copa Courier, Correos de Cuba, Aerovaradero y DHL.
📢 Informacion Estrategica: Seccion de noticias y avisos importantes sobre regulaciones de aduana y estados de carga.
⚡ Optimizacion de Carga: Interfaz moderna construida con Next.js 15 y Tailwind CSS.
📱 Diseno Responsivo: Completamente adaptado para dispositivos moviles y escritorio.

🛠 Stack Tecnologico
💻 Framework: Next.js (App Router)
⌨️ Lenguaje: TypeScript
🎨 Estilos: Tailwind CSS
🔠 Fuentes: Geist Sans y Mono (Vercel)

🏁 Configuracion del Proyecto
📋 Requisitos previos
Asegurate de tener instalado Node.js (version 18 o superior).

⚙️ Instalacion
Clona el repositorio:
git clone https://github.com/ajliy97/La-Herradura-SPA.git

Instala las dependencias:
npm installs

💻 Desarrollo
Inicia el servidor de desarrollo:

npm run dev

o
yarn dev

o
pnpm dev

o
bun dev

Abre http://localhost:3001 con tu navegador para ver el resultado. Puedes empezar a editar la pagina modificando app/page.tsx. La pagina se actualiza automaticamente al guardar los cambios.

📂 Estructura del Proyecto
📁 app/: Contiene las rutas y la logica principal del App Router.
📁 components/: Componentes reutilizables como el buscador de rastreo y selectores de sitio.
📁 public/: Imagenes de marcas de courier y activos estaticos.
📁 styles/: Configuraciones globales de CSS y Tailwind.
